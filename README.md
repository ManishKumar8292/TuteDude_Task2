# Laundry Wallah Website

This is a simple HTML project that displays a basic website for a laundry service called **Laundry Wallah / Laundry Mart**. The webpage provides information about laundry services, pricing, and allows users to enter their contact details for booking.

## 📌 Project Description
The website is created using **HTML5** and demonstrates the basic structure of a webpage including header, sections, images, tables, forms, and a footer. It is designed to show how a small laundry business can present its services online.

## 🛠 Technologies Used
- HTML5

## ▶️ How to Run the Project
1. Download or clone the repository.
2. Open the project folder.
3. Double click on **index.html** or open it in any web browser.
